# xwguidecompress-图形化解压缩软件

使用 Python 制作的图形化解压缩软件，**对比目前市面上的成熟软件没有任何优势**，**并且可能存在很多问题**。由希望Windows的hh开发，本人项目前均有“xw”作为前缀

## 快速开始

#### 前提条件

- **Python 3.6+**
- **Windows**: Windows 7 SP1 及以上
- **Linux**: Ubuntu 16.04+ / Debian 9+ / Fedora 26+ 或任何能安装 Python 3.6 的发行版
- **macOS**: 10.9 Mavericks 及以上

## 使用方法

克隆项目

```
git clone https://github.com/xwWindowsdhh/xwguidecompress
cd xwguidecompress
```

运行项目

```
python.exe src/main.py
```
